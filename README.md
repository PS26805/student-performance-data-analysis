# student-performance-data-analysis
Data cleansing, exploratory data analysis (EDA), and visualizations are all included in the examination of student performance data using Python, Pandas, Matplotlib, and Seaborn.
# Project Overview
This project analyzes the Student Performance dataset using Python. It includes data cleaning, exploratory data analysis (EDA), and data visualization to gain insights into students' academic performance.

# Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

# Tasks Performed
- Loaded the dataset
- Cleaned the data
- Checked for missing values and duplicates
- Performed exploratory data analysis (EDA)
- Created data visualizations
- Analyzed student performance
- Wrote conclusions

# Results
- Average Final Grade (G3): *10.42*
- Students scoring above 15: *40*
- Very weak positive correlation between study time and final grades
- Male students had a slightly higher average final grade than female students

# Conclusion
This project demonstrates how Python can be used to clean, analyze, and visualize educational data. The analysis provides useful insights into factors related to student performance.
